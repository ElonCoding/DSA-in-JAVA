# DSA-in-JAVA

This repository contains various Java programs covering fundamental programming concepts and data structures and algorithms (DSA).

## Programs Implemented:

### Day 1:
- **EvenOdd.java**: Checks if a given integer is even or odd using an if-else statement.
- **GreatestOfTwo.java**: Finds the greatest of two numbers using an if-else statement.
- **PositiveNegativeZero.java**: Checks if a given integer is positive, negative, or zero.
- **Print1ToN.java**: Prints numbers from 1 to N using a for loop.
- **SumOfNNumbers.java**: Calculates the sum of the first N natural numbers using a while loop.
- **VowelConsonant.java**: Checks whether a character is a vowel or a consonant using a switch-case statement.

## Getting Started

To compile and run these Java programs, you need to have the Java Development Kit (JDK) installed and configured in your system's PATH environment variable.

### Compilation:
Navigate to the directory containing the `.java` file and compile it using the Java compiler:
```bash
javac YourProgramName.java
```

### Execution:
After successful compilation, run the program using the Java Virtual Machine (JVM):
```bash
java YourProgramName
```
